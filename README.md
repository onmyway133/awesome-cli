## Quick Install (macOS)

```bash
# Core modern replacements
brew install bat eza ripgrep fd zoxide fzf

# Git and file management
brew install lazygit delta yazi difftastic

# System monitoring
brew install btop bottom bandwhich

# Shell enhancement
brew install starship atuin fish

# Developer tools
brew install just jq yq httpie mise uv

# Database clients
brew install pgcli mycli litecli

# TUI frameworks
brew install gum glow

# Network tools
brew install trippy doggo gping

# Media
brew install yt-dlp mpv
```
