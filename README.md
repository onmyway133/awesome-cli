## Quick Install (macOS)

```bash
# Core modern replacements
brew install bat eza ripgrep fd zoxide fzf

# Git and file management
brew install lazygit delta yazi

# System monitoring
brew install btop bottom

# Shell enhancement
brew install starship atuin

# Developer tools
brew install just jq yq httpie
```
